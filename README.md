# wordle-game-clone

## game-api-route

|path| verb| Comment |
|----------------|-----|-------------------------|
|-|||
|/api/word| post | {word: [["w"], ["o"],["r"], ["d"]], id: uuid} |
|-|||
|/api/word| get | get random word |
|/api/word?length=Number| get | get random word with length === Num |
|/api/word?mulitChar=false| get | get random word with 1 repeated char|
